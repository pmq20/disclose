# Disclose

Pack your Node.js project into an executable without compiling.

## Installation

    $ gem install disclose

## Usage

    $ disclose <node_path> <project_path>

## Example

    $ disclose /usr/local/bin/node /usr/local/lib/node_modules/coffee-script
